Introduction

The goal of this project is to create an interactive application allowing users to view, interact with, and gain insight into the Sacramento Police Department Dispatch Data available on the City of Sacramento Open Data Portal. While it is fantastic that the city makes the data publically available, it is very difficult to gain any insight into the data, especially someone who may not be technically inclined. Thus, hopefully my app will be able to bridge the gap between the data and the user.

This project is currently under development. The code for the project is available on GitHub.
